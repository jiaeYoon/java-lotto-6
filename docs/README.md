### 로또 기능 명세서
1. 로또 구입 금액 입력
   1. 숫자인지 확인
   2. 1,000으로 나누어 떨어지는지 확인
2. 로또 발행
3. 발행한 로또 수량 및 번호 출력
4. 당첨 번호 입력
   1. 6개 입력했는지 확인
   2. 숫자인지 확인
   3. 숫자 범위(1~45) 확인
   4. 중복 숫자 확인
5. 보너스 번호 입력
   1. 숫자인지 확인 
   2. 숫자 범위(1~45) 확인
   3. 중복 숫자 확인
6. 당첨 내역 계산
7. 당첨 내역 출력
8. 수익률 출력

----------------
**클래스**
로또 - 로또 발행
당첨 - 당첨 번호, 보너스 번호, 당첨 계산, (수익률)

**변수**
금액 - 로또 발행, 수익률 계산
